### 3.2 Создание проекта

Создана папка `lab2`, в ней файл `main.py` со следующим кодом:

python
def calculate_average(numbers):
    total = 0
    for i in range(len(numbers)):
        total += numbers[i]
    return total / len(numbers)

nums = [10, 20, 30]
print(calculate_average(nums))
